  • path(normalized:)

    Creates a path from the collection of GeoCoordinates. Useful if you want to draw a 2D image of a track.

    Declaration

    Swift

    func path(normalized: Bool = true) -> CGPath

    Parameters

    normalized

    Flag indicating if the paths values should be normalized into the range 0…1. If true, the resulting values in the path are mapped to value in 0…1 coordinates space, otherwise the values from the geo coordinates. Defaults to true.

    Return Value

    A CGPath containg a projected 2D-representation of the geo coordinates.

    Return Value

    An array of Coordinate values, each having a minimum distance to their predecessors of least closerThan meters.
