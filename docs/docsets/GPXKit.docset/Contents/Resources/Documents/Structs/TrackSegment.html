TrackSegment

public struct TrackSegment : Hashable

Value type descriping a logical segment in a TrackGraph. A TrackGraph consists of a collection of TrackSegments. Each has a coordinate (latitude, longitude & elevation) and the distance (in meters) to its preceding segment point.
